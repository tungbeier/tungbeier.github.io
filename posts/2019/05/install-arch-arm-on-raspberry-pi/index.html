Install Arch Arm on Raspberry Pi

  1. Start fdisk to partition the SD card:

    fdisk /dev/sdX

  2. At the fdisk prompt, delete old partitions and create a new one:

    • Type o to clear out any partitions on the drive.
    • Type p to list partitions. There should be no partitions left.
    • Type n, then p for primary, 1 for the first partition on the drive, press ENTER to accept the default first sector, then type +100M for the last sector.
    • Type t, then c to set the first partition to type W95 FAT32 (LBA).
    • Type n, then p for primary, 2 for the second partition on the drive, and then press ENTER twice to accept the default first and last sector.
    • Write the partition table and exit by typing w.

  3. Create and mount the FAT filesystem:

    mkfs.vfat /dev/sdX1
mkdir boot
mount /dev/sdX1 boot

  4. Create and mount the ext4 filesystem:

    mkfs.ext4 /dev/sdX2
mkdir root
mount /dev/sdX2 root

  5. Download the ArchLinuxARM tar:

    wget http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-<Raspberry-Pi-version>-latest.tar.gz

  6. Change to root and extract the root filesystem (do not use sudo):

    su
bsdtar -xpf ArchLinuxARM-rpi-<Raspberry-Pi-version>-latest.tar.gz -C root
sync

  7. Move boot files to the first partition:

    mv root/boot/* boot

  8. Unmount the two partitions:

    umount boot root

  9. Insert the SD card into the Raspberry Pi and boot up.

  10. Login as the default user alarm with the password alarm. The default root password is root.

  11. Initialize the pacman keyring and populate the Arch Linux ARM package signing keys:

    pacman-key --init
pacman-key --populate archlinuxarm
pacman -Syyuu
reboot

  12. Set the hostname of your Pi as per your preference

    hostnamectl set-hostname YOUR_HOST_NAME

    Adjust the hosts file by adding the following

    127.0.0.1   localhost.localdomain       localhost
::1         localhost.localdomain       localhost
127.0.0.1   YOUR_HOST_NAME.localdomain  YOUR_HOST_NAME

  13. Set hardware clock to UTC and set timezone (here "Europe/Berlin")

    timedatectl set-local-rtc 0
ln -sf /usr/share/zoneinfo/Europe/Berlin /etc/localtime

  14. NTP

    pacman -S ntp fake-hwclock
systemctl enable ntpd.service
systemctl start ntpd.service

  15. User setup
    Change the password for root

    passwd

    Install sudo and vim

    pacman -S sudo vim

    Enable the wheel group, create your user, and add the user to the group.

    visudo

    Search for the line contains %wheel ALL=(ALL) ALL. Uncomment it and save the file.

    Create your user, set your password and delete the default user.

    useradd -m -G wheel -s /bin/bash YOURUSERNAME
passwd YOURUSERNAME
userdel alarm

    Log out and log back in as the new user.

  16. Setup swap (Optional)

    fallocate -l 1024M /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo 'vm.swappiness=1' > /etc/sysctl.d/99-sysctl.conf

    Configure fstab

    /swapfile none swap defaults 0 0

  17. Increase the life of the SD card, add the following to /etc/fstab

    /dev/root  /  ext4  defaults,nodiratime,noatime,discard  0  0
