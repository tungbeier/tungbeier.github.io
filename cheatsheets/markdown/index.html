Lists

Ordered one (the numbers don't matter)

1. First item
1. Second item
  1. Sub-Item

Unordered one can use asterisks(*), minus(-) or plus(+)

* First item
- Second item
+ Third item
  * Sub-Item

Links

[link](link url)

[link with title](link url "title")

[link][reference as text or number]
[reference as text or number]: url

Images

![alt text](url to image "title")

![alt text][reference]
[reference]: url to image

Tables

There must be at least 3 dashes separating each header cell.
The outer pipes (|) and lining up the raw markdown are both optional.
Inline Markdown can also be used in the columns.
Colons (:) can be used to align columns.

| Table  | Header   |         |         |
| ------ |:--------:| :------ | ------: |
| col 1  | centered | left-   | right-  |
| col 2  | column   | aligned | aligned |
